# What is an API Key?

An API key is a way for a service to determine who you are and whether you have access to this service

It is kind of like a login, only this time, there is no username and password!

# Why would I need a Google Gemini API key?

The program requires access to Google Gemini to create specifications usable by machines. To use Gemini outside of the website (such as using it here), it would need to get an API key to determine who you are.

The main reason why I am not giving you my API key is because of privacy reasons and the fact that the free tier of Google Gemini is only restricted to 20 messages a day.

# Where is my API key being used?

Your API key is only used during the initial setup. IT IS NEVER STORED ON YOUR COMPUTER.

# How to get an API Key?

1. Open this link here: https://ai.google.dev/gemini-api/docs/api-key
2. Click on the blue button that says "Create or view a Gemini API Key"

   <img width="535" height="674" alt="image" src="https://github.com/user-attachments/assets/47ad1acd-5d3c-4cf5-bc2f-30c801cc0f4e" />

3. Sign in to Google (if you already have not)

4. Then go to projects:

   <img width="267" height="451" alt="image" src="https://github.com/user-attachments/assets/a9c7adce-49f5-40f1-bee4-3041ea5ab4c8" />
   
5. Click on "Create new project":

   <img width="1099" height="176" alt="image" src="https://github.com/user-attachments/assets/64f2ab38-4534-4a19-b296-00c534f5fd00" />
   
6. Name your project (doesn't matter what it is called)
7. Then go to API Keys
8. Click on create API Key:

   <img width="1109" height="176" alt="image" src="https://github.com/user-attachments/assets/458ee4f0-f2dc-46d4-af7c-954193b40a0b" />
   
9. Name your API Key and choose the project you just created:

<img width="621" height="334" alt="image" src="https://github.com/user-attachments/assets/1bbf6790-f2b5-4722-bbc8-ab709c0928e9" />
   
10. Click on "Create key"
11. Copy your new API key!

Now you obtained your Google Gemini API key and can finally use it in Beevee!
